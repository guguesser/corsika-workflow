Firstly, send an email to tanguy.pierog@kit.edu expressing interest in using the software so that he can provide the password required for the program installation.
The installation of CORSIKA77500 and all its folders is available at the following link: https://web.iap.kit.edu/corsika/download/
Enter the username 'corsika' and the password provided through the email received from the software's technical team.
corsika-v770 > corsika-77500.tar.gz to get the most recent version of corsika-77500.

As an example, if you have the file corsika-77500.tar.gz in the current directory, the installation on a LINUX system will look like (the commands are highlighted with the symbol '>'):
> tar -zxf corsika-77500.tar.gz
> cd corsika-77500/
> ./coconut
> Compile in 32 or 64bit mode? [2 - Use compiler default ('-m64' on a 64bit machine)]
> Which high energy hadronic interaction model do you want to use? [2 -  EPOS LHC]
> Which low energy hadronic interaction model do you want to use? [4 - URQMD 1.3cr]
> Which detector geometry do you have? [1 - horizontal flat detector array]
> Which additional CORSIKA program options do you need? [8a - shower PLOT version (PLOTSH) (only for single events) and 8b - shower PLOT(C) version (PLOTSH2) (only for single events)]
> z - *** Finish selection ***
> Are you sure you want to continue with these current option selection: PLOTSH2 PLOTSH
  yes or no ? (default: yes) > yes
> Configuration is finished. How do you want to proceed? [f - Compiling and remove temporary files]
